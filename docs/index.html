Widget v1

Pick a hero colour, get a complete UI. Saturated monochromatic theming, tactile shadows, soft-square radii. Zero dependencies.

1 <link rel="stylesheet" href="widget.css">
2 <script src="widget-theme.js"></script>

Theme System
One hero colour generates everything. Drag the slider — every component below updates live.
Hero Hue
Saturation %
Lightness %
Presets
/* Option 1: CSS var (set before scripts) */
<style>:root { --wg-hero-hex: #1093af; }</style>

/* Option 2: JavaScript */
WidgetTheme.set(240);               // hue integer
WidgetTheme.set('#1093af');        // hex
WidgetTheme.set('hsl(200,60%,37%)'); // hsl string
WidgetTheme.set(240, 55, 42);       // hue, sat%, light%

/* Presets from original apps */
WidgetTheme.presets.pomodoro();     // → red
WidgetTheme.presets.calculator();   // → teal
WidgetTheme.presets.todo();         // → lime

/* Override individual tokens */
WidgetTheme.override('--wg-btn', '#ff0000');
